Ejercicio 1

Ejercicio1: Ve a la web www.aliexpress.com y abre la consola. Recupera la URL y reemplaza el primer .com por un .es y pásala a mayúsculas. ¿En qué posición empieza el último ES que aparece?

Solución

    //La URL es https://es.aliexpress.com/?gatewayAdapt=glo2esp
    let u = location.href;
    let pos = u.replace("com", "es").toUpperCase().lastIndexOf("ES");
    console.log(`La posición del último ES es ${pos}`);

Ejercicio 2

Ejercicio 2: partiendo de "caca de la vaca" crea una cadena que tenga un - entre cada letra. Es decir: c-a-c-a- -d-e- -l-a- -v-a-c-a

Solución

    let s = "caca de la vaca";
    s.replaceAll("","-").slice(1,-1);

Ejercicio 3

Ejercicio 3: Partiendo del navigator.userAgent suma todos los números que aparecen en la cadena de caracteres.

Por ejemplo si el userAgent es Firefox/117.0 el resultado será 9 (1+1+7+0).

Nota: tal vez las funciones isNaN() y Number() puedan serte de utilidad.

Solución

    let userAgent = navigator.userAgent;
    sol=0;
    for(let i = 0; i<userAgent.length;i++){
      if( !isNaN( userAgent[i] )  ){
        sol+=Number(userAgent[i]);
      }
    }
    console.log("La solución es " + sol);
